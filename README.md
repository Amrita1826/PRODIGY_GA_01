# 🎨 DALL·E Mini Image Generator (Task 2)

This project demonstrates how to generate images from text prompts using a **DALL·E Mini–style model** in **Google Colab**. It showcases the application of generative AI techniques to convert natural language descriptions into visual outputs.

---

## 🚀 Project Overview

The goal of this project is to:

- Generate images from textual prompts  
- Understand the working of text-to-image models  
- Experiment with generative AI using an easy-to-use notebook environment  

The implementation is done in **Google Colab**, making it accessible without requiring local setup.
