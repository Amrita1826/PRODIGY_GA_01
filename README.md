# 🎓 Fine-Tuning GPT-2 on NYU Q&A Dataset (Task 1)

This project demonstrates how to **fine-tune OpenAI's GPT-2 language model** using a custom **NYU (New York University) Question & Answer dataset** in **Google Colab**.

The trained model learns to generate relevant answers based on user prompts related to NYU.

---

## 🚀 Project Overview

The main objectives of this project are:

- Fine-tune a pre-trained GPT-2 model on a custom dataset  
- Train the model using NYU-related Question & Answer pairs  
- Generate meaningful answers from prompts  
- Understand NLP model fine-tuning workflow using Hugging Face Transformers  
- Run the complete project easily on Google Colab
